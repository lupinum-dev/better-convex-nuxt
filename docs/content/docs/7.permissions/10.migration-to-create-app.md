---
title: 'Migration To createApp'
description: 'Replace the removed transitional APIs with the hard-cut createApp surface.'
---

## Replacement Map

- `createFunctions(query, mutation, options)` plus `defineHandler(...)`
  Replace with `createApp(query, mutation, options)`.
- `appQuery(...)`
  Replace with `app.query(...)` for canonical handlers or `raw.query(...)` for explicit escape hatches.
- `appMutation(...)`
  Replace with `app.mutation(...)` for canonical handlers or `raw.mutation(...)` for explicit escape hatches.
- `definePermissions(...)`
  Replace with `definePermissionContext(...)`.
- `withCan(resource, map)`
  Replace with `defineCapabilities(...).attach(actor, resource)`.
- `createDefaultGetActor`, `defineActorExtension`, `defineActorFromMembership`
  Replace with `defineActor.fromAuth()` / `.fromMembership()` / `.extend()` / `.filter()`.

## Before

```ts
import { createFunctions, defineHandler } from '@lupinum/trellis/functions'

export const { query: appQuery, mutation: appMutation } = createFunctions(query, mutation, {
  actor: getActor,
})

export const app = defineHandler(appQuery, appMutation)
```

## After

```ts
import { createApp } from '@lupinum/trellis/functions'

export const { app, raw } = createApp(query, mutation, {
  actor: getActor,
})
```

## Before

```ts
export const list = appQuery({
  args: {},
  handler: async (ctx) => {
    const actor = await ctx.actor()
    enforce(actor, 'Read todos', canReadTodo)
    return await ctx.db.query('todos').collect()
  },
})
```

## After

```ts
export const list = app.query({
  guard: canReadTodo,
  args: {},
  handler: async (ctx) => {
    return await ctx.db.query('todos').collect()
  },
})
```

## Raw Escape Hatch

If the handler genuinely needs the lower-level path, make that explicit:

```ts
export const generateUploadUrl = raw.mutation({
  args: {},
  handler: async (ctx) => {
    const actor = await ctx.actor()
    enforce(actor, 'Generate upload URL', isAuthenticated)
    return await ctx.storage.generateUploadUrl()
  },
})
```
