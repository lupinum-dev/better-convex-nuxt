## Recommended Order

1. Read [Start Here](/docs/getting-started/start-here).
2. Complete [Installation](/docs/getting-started/installation).
3. Do [First Live Query](/docs/getting-started/first-live-query).
4. Move to [Build a Signed-In Todo App](/docs/getting-started/build-a-signed-in-todo-app) only when you want the protected path.
