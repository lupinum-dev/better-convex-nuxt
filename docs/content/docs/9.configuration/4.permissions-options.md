Use this page to look up `trellis.permissions`.

## Shape

Two supported forms:

```ts
permissions: 'workspaces.getPermissionContext'
```

```ts
permissions: {
  query: 'workspaces.getPermissionContext',
}
```

## What It Does

This option wires the built-in:

- `usePermissions()`
- `useAuthGuard()`

to one app-owned Convex query that returns the frontend permission context.

## Important Constraint

The string must point at a real Convex query export path. If it does not, Trellis raises a configured-function error instead of silently doing nothing.
