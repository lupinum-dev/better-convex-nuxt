Use this page to look up `trellis.mcp`.

## Shape

```ts
mcp: {
  name: 'My App',
  sessions: false,
}
```

## Options

- `name`: name shown to MCP clients
- `sessions`: enable MCP session state

## Scope

These are module-level defaults and enablement flags. Tool definition, runtime auth, and operation binding live in the MCP docs and API reference.
