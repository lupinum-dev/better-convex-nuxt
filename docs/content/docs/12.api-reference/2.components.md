## ConvexAuthenticated

Renders its slot content only when the user is authenticated and the auth check is complete.

```vue
<ConvexAuthenticated>
  <p>Welcome back!</p>
</ConvexAuthenticated>
```

**Behavior:**
- Hidden while auth is pending
- Hidden when user is not authenticated
- Rendered when `isAuthenticated` is `true` and `isPending` is `false`

---

## ConvexUnauthenticated

Renders its slot content only when the user is **not** authenticated and the auth check is complete.

```vue
<ConvexUnauthenticated>
  <p>Please sign in to continue.</p>
</ConvexUnauthenticated>
```

**Behavior:**
- Hidden while auth is pending
- Hidden when user is authenticated
- Rendered when `isAuthenticated` is `false` and `isPending` is `false`

---

## ConvexAuthLoading

Renders its slot content while the authentication state is being determined.

```vue
<ConvexAuthLoading>
  <div class="spinner">Checking authentication...</div>
</ConvexAuthLoading>
```

**Behavior:**
- Rendered when `isPending` is `true`
- Hidden once auth state settles (authenticated or not)

---

## ConvexAuthError

Renders its slot content when authentication fails, with a scoped slot providing error details and a retry function.

```vue
<ConvexAuthError v-slot="{ error, retry }">
  <div class="error">
    <p>Authentication failed: {{ error }}</p>
    <button @click="retry">Try Again</button>
  </div>
</ConvexAuthError>
```

**Slot Props:**

| Prop | Type | Description |
|------|------|-------------|
| `error` | `string \| null` | The auth error message |
| `retry` | `() => void` | Function to retry authentication |

---

## Combining Components

Use all four components together for complete auth state coverage:

```vue
<template>
  <ConvexAuthLoading>
    <LoadingSpinner />
  </ConvexAuthLoading>

  <ConvexAuthenticated>
    <AppDashboard />
  </ConvexAuthenticated>

  <ConvexUnauthenticated>
    <LoginPage />
  </ConvexUnauthenticated>

  <ConvexAuthError v-slot="{ error, retry }">
    <ErrorDisplay :message="error" @retry="retry" />
  </ConvexAuthError>
</template>
```

::note
All auth components are auto-imported when `auth: true` is set in your module config.
::
