Import server helpers from `better-convex-nuxt/server`.

```ts
import {
  serverConvexQuery,
  serverConvexMutation,
  serverConvexAction,
} from 'better-convex-nuxt/server'
```

Shared schema helpers live under `better-convex-nuxt/schema`, and MCP-only helpers live under `better-convex-nuxt/mcp`. This split is intentional so Nitro server code does not import the client-heavy composables entrypoint.

## serverConvexQuery

Run a Convex query from a Nuxt server route.

```ts
// Auto-resolve event (requires experimental.asyncContext)
async function serverConvexQuery<Query>(
  query: Query,
  args?: FunctionArgs<Query>,
  options?: ServerConvexOptions
): Promise<FunctionReturnType<Query>>

// Explicit event
async function serverConvexQuery<Query>(
  event: H3Event,
  query: Query,
  args?: FunctionArgs<Query>,
  options?: ServerConvexOptions
): Promise<FunctionReturnType<Query>>
```

**Example:**

```ts [server/api/tasks.get.ts]
export default defineEventHandler(async (event) => {
  return await serverConvexQuery(event, api.tasks.list, { status: 'active' })
})
```

## serverConvexMutation

```ts
async function serverConvexMutation<Mutation>(
  mutation: Mutation,
  args?: FunctionArgs<Mutation>,
  options?: ServerConvexOptions
): Promise<FunctionReturnType<Mutation>>

async function serverConvexMutation<Mutation>(
  event: H3Event,
  mutation: Mutation,
  args?: FunctionArgs<Mutation>,
  options?: ServerConvexOptions
): Promise<FunctionReturnType<Mutation>>
```

**Example:**

```ts [server/api/tasks.post.ts]
export default defineEventHandler(async (event) => {
  const body = await readBody(event)
  return await serverConvexMutation(event, api.tasks.create, { text: body.text })
})
```

## serverConvexClearAuthCache

Clear a cached auth token for a session. This is mainly for custom server-side revocation or logout flows. Ordinary `useConvexAuth().signOut()` already clears the current session's cached JWT automatically.

```ts
async function serverConvexClearAuthCache(sessionToken: string): Promise<void>
```

**Example:**

```ts
const sessionToken =
  getCookie(event, '__Secure-better-auth.session_token')
  ?? getCookie(event, 'better-auth.session_token')
if (sessionToken) {
  await serverConvexClearAuthCache(sessionToken)
}
```

## ServerConvexOptions

Options shared by all server utilities:

| Option | Type | Default | Description |
|--------|------|---------|-------------|
| `auth` | `'auto' \| 'required' \| 'none'` | `'auto'` | Authentication mode |
| `authToken` | `string` | — | Explicit JWT token override |

**Auth modes:**

| Mode | Behavior |
|------|----------|
| `'auto'` | Uses session cookie if available, runs without auth otherwise |
| `'required'` | Throws if no auth token can be resolved |
| `'none'` | Skips authentication entirely |
