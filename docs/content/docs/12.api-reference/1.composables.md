## useConvexQuery

Fetch real-time data from Convex with SSR support.

```ts
function useConvexQuery<Query, Args, DataT>(
  query: Query,
  args?: MaybeRefOrGetter<Args | null | undefined>,
  options?: UseConvexQueryOptions<RawT, DataT>
): UseConvexQueryReturn<DataT>
```

**Return Value:**

| Property | Type | Description |
|----------|------|-------------|
| `data` | `Ref<DataT \| null>` | Query result |
| `status` | `Ref<QueryStatus>` | `'pending'` \| `'success'` \| `'error'` \| `'skipped'` |
| `pending` | `Ref<boolean>` | `true` while loading |
| `error` | `Ref<Error \| null>` | Error if failed |
| `refresh()` | `() => Promise<void>` | Re-fetch the query |
| `clear()` | `() => void` | Reset to initial state |

**Options:**

| Option | Type | Default | Description |
|--------|------|---------|-------------|
| `server` | `boolean` | `true` | Run during SSR |
| `subscribe` | `boolean` | `true` | Keep WebSocket subscription |
| `default` | `() => RawT` | — | Fallback while pending |
| `transform` | `(input: RawT) => DataT` | — | Transform raw data |
| `keepPreviousData` | `boolean` | `false` | Preserve previous data while loading |
| `shared` | `string` | — | Share state across components |
| `onData` | `(data: DataT) => void` | — | Called on new data |
| `onError` | `(error: Error) => void` | — | Called on error |

## useConvexPaginatedQuery

Load data incrementally with pagination.

```ts
function useConvexPaginatedQuery<Query, Args, TransformedItem>(
  query: Query,
  args?: MaybeRefOrGetter<Args | null | undefined>,
  options?: UseConvexPaginatedQueryOptions<Item, TransformedItem>
): UseConvexPaginatedQueryReturn<TransformedItem>
```

**Return Value:**

| Property | Type | Description |
|----------|------|-------------|
| `results` | `ComputedRef<Item[]>` | All loaded items |
| `status` | `ComputedRef<PaginatedQueryStatus>` | Status state machine |
| `isLoading` | `ComputedRef<boolean>` | Loading first page or more |
| `isExhausted` | `ComputedRef<boolean>` | All pages loaded |
| `hasNextPage` | `ComputedRef<boolean>` | More pages available |
| `loadMore(n)` | `(numItems: number) => void` | Load next page |
| `error` | `Ref<Error \| null>` | Error if failed |
| `refetch()` | `() => Promise<void>` | Refresh all loaded pages |
| `restart()` | `() => Promise<void>` | Reset to first page |

**Status values:** `'skipped'` | `'loading-first-page'` | `'ready'` | `'loading-more'` | `'exhausted'` | `'error'`

**Options:**

| Option | Type | Default | Description |
|--------|------|---------|-------------|
| `initialNumItems` | `number` | `10` | Items per page |
| `server` | `boolean` | `true` | Run first page during SSR |
| `subscribe` | `boolean` | `true` | Keep subscriptions |
| `default` | `() => Item[]` | — | Fallback items |
| `transform` | `(items: Item[]) => TransformedItem[]` | — | Transform items |
| `keepPreviousData` | `boolean` | `false` | Preserve previous results |

## useConvexMutation

Call Convex mutations with state tracking and optimistic updates.

```ts
function useConvexMutation<Mutation>(
  mutation: Mutation,
  options?: UseConvexMutationOptions<Args, Result>
): UseConvexMutationReturn<Args, Result>
```

**Return Value:** A callable function `(args) => Promise<Result>` with:

| Property | Type | Description |
|----------|------|-------------|
| `data` | `Ref<Result \| undefined>` | Last successful result |
| `status` | `ComputedRef<MutationStatus>` | `'idle'` \| `'pending'` \| `'success'` \| `'error'` |
| `pending` | `ComputedRef<boolean>` | `true` while in progress |
| `error` | `Ref<Error \| null>` | Error from last attempt |
| `reset()` | `() => void` | Reset to idle |

**Options:**

| Option | Type | Description |
|--------|------|-------------|
| `optimisticUpdate` | `(ctx: OptimisticContext, args: Args) => void` | Optimistic update callback |
| `onSuccess` | `(result: Result, args: Args) => void` | Called after success |
| `onError` | `(error: Error, args: Args) => void` | Called after failure |

## useConvexAction

Call Convex actions with state tracking.

```ts
function useConvexAction<Action>(
  action: Action,
  options?: { onSuccess?, onError? }
): UseConvexMutationReturn<Args, Result>
```

Same return value as `useConvexMutation`, without the `optimisticUpdate` option.

## useConvexUpload

Upload files to Convex storage.

```ts
function useConvexUpload<Mutation>(
  generateUploadUrlMutation: Mutation,
  options?: UseConvexUploadOptions
): UseConvexUploadReturn<Mutation>
```

**Return Value:** A callable function `(input: File | File[]) => Promise<string | string[]>` with:

| Property | Type | Description |
|----------|------|-------------|
| `upload(input, args?)` | `Function` | Upload file(s) |
| `data` | `Ref<string \| undefined>` | Last single-file storageId |
| `status` | `ComputedRef<UploadStatus>` | `'idle'` \| `'pending'` \| `'success'` \| `'error'` |
| `pending` | `ComputedRef<boolean>` | Upload in progress |
| `progress` | `ComputedRef<number>` | 0–100 progress |
| `error` | `Ref<Error \| null>` | Upload error |
| `items` | `Ref<UploadQueueItem[]>` | Queue items (queue mode) |
| `cancelItem(id)` | `(id: string) => void` | Cancel specific item |
| `cancelAll()` | `() => void` | Cancel all uploads |
| `clearFinished()` | `() => void` | Remove finished items |
| `reset()` | `() => void` | Reset everything |

## useConvexStorageUrl

Generate a URL for a file in Convex storage.

```ts
function useConvexStorageUrl(
  query: FunctionReference<'query'>,
  storageId: MaybeRef<string | null | undefined>
): ComputedRef<string | null>
```

Auto-skips when `storageId` is `null` or `undefined`.

## useConvexAuth

Access authentication state.

```ts
function useConvexAuth(): UseConvexAuthReturn
```

| Property | Type | Description |
|----------|------|-------------|
| `user` | `Readonly<Ref<ConvexUser \| null>>` | Authenticated user |
| `isAuthenticated` | `ComputedRef<boolean>` | Whether authenticated |
| `isPending` | `Readonly<Ref<boolean>>` | Auth check in progress |
| `signOut()` | `() => Promise<void>` | Sign out user |

::note
Only available when `auth: true` is set in config.
::

---

## useConvexAuthInternal

Access internal auth state for advanced use cases.

```ts
function useConvexAuthInternal(): UseConvexAuthInternalReturn
```

| Property | Type | Description |
|----------|------|-------------|
| `token` | `Readonly<Ref<string \| null>>` | Raw JWT token |
| `authError` | `Readonly<Ref<string \| null>>` | Auth error message |
| `refreshAuth()` | `() => Promise<void>` | Force token refresh |
| `awaitAuthReady(options?)` | `(opts?: { timeoutMs?: number }) => Promise<boolean>` | Wait for auth to settle |

## useConvexConnectionState

Monitor WebSocket connection status.

```ts
function useConvexConnectionState(): {
  state, isConnected, isReconnecting, pendingMutations, shouldShowOfflineUi
}
```

| Property | Type | Description |
|----------|------|-------------|
| `state` | `Readonly<Ref<ConnectionState>>` | Full connection state |
| `isConnected` | `ComputedRef<boolean>` | WebSocket connected |
| `isReconnecting` | `ComputedRef<boolean>` | Was connected, now reconnecting |
| `pendingMutations` | `ComputedRef<number>` | Mutations waiting |
| `shouldShowOfflineUi` | `ComputedRef<boolean>` | Show offline UI (with grace period) |

## useConvex

Access the raw Convex client instance.

```ts
function useConvex(): ConvexClient
```

Returns the singleton `ConvexClient` instance. Client-only.

## createPermissions

Factory for creating permission composables.

```ts
function createPermissions<TPermission, TContext>(options: {
  query: FunctionReference<'query'>,
  checkPermission: CheckPermissionFn<TPermission>
}): {
  usePermissions: () => UsePermissionsReturn<TPermission, TContext>,
  usePermissionGuard: (options: UsePermissionGuardOptions<TPermission>) => void
}
```

::note
Only available when `permissions: true` is set in config.
::

## executeConvexQuery

One-shot query execution (no composable state).

```ts
async function executeConvexQuery<Query>(
  query: Query,
  args: Args,
  options?: { subscribe?: boolean }
): Promise<Result>
```
