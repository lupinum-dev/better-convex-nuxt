Use this page when you need the Nuxt server-side entrypoints.

## Main Exports

- `serverConvexQuery`
- `serverConvexMutation`
- `serverConvexAction`
- `createServerConvexCaller`

## Auth Modes

These helpers support:

- `auto`
- `required`
- `none`
- `trusted`

## Notes

- request-scoped auth resolution is shared per Nitro request
- `createServerConvexCaller(...)` is a convenience wrapper, not a second authorization model
