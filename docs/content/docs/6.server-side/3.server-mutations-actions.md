## Overview

`serverConvexMutation` and `serverConvexAction` let you write data and run server-side logic from Nuxt server routes, webhooks, and API endpoints.

## Server Mutations

```ts [server/api/tasks.post.ts]
import { api } from '~~/convex/_generated/api'

export default defineEventHandler(async (event) => {
  const body = await readBody(event)

  const taskId = await serverConvexMutation(event, api.tasks.create, {
    text: body.text,
    status: 'active',
  })

  return { id: taskId }
})
```

## Server Actions

```ts [server/api/reports/generate.post.ts]
import { api } from '~~/convex/_generated/api'

export default defineEventHandler(async (event) => {
  const body = await readBody(event)

  const report = await serverConvexAction(event, api.reports.generate, {
    startDate: body.startDate,
    endDate: body.endDate,
  })

  return report
})
```

## Call Signatures

Both functions support the same two signatures as `serverConvexQuery`:

```ts
// Auto-resolve event
serverConvexMutation(mutation, args?, options?)
serverConvexAction(action, args?, options?)

// Explicit event
serverConvexMutation(event, mutation, args?, options?)
serverConvexAction(event, action, args?, options?)
```

## Options

Same `ServerConvexOptions` as server queries:

::field-group
::field{name="auth" type="'auto' | 'required' | 'none'"}
Authentication mode. Default: `'auto'`.
::
::field{name="authToken" type="string"}
Explicit JWT token override.
::
::

## Use Cases

### Webhook Handler

```ts [server/api/webhooks/stripe.post.ts]
import { api } from '~~/convex/_generated/api'

export default defineEventHandler(async (event) => {
  const body = await readBody(event)

  // Webhooks don't have user auth — use 'none'
  await serverConvexMutation(
    event,
    api.payments.recordPayment,
    { stripeEventId: body.id, amount: body.amount },
    { auth: 'none' }
  )

  return { received: true }
})
```

### Form Submission

```ts [server/api/contact.post.ts]
import { api } from '~~/convex/_generated/api'

export default defineEventHandler(async (event) => {
  const body = await readBody(event)

  await serverConvexMutation(event, api.messages.create, {
    name: body.name,
    email: body.email,
    message: body.message,
  }, { auth: 'none' })

  return { success: true }
})
```
