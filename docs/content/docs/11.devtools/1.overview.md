## Overview

Better Convex Nuxt integrates with [Nuxt DevTools](https://devtools.nuxt.com/) to provide a dedicated "Convex" tab for inspecting queries, mutations, auth state, and connection status.

## Features

### Query Inspector

View all active queries with their:
- Function name and arguments
- Current status (pending, success, error)
- Data source (SSR HTTP or WebSocket subscription)
- Live data preview

### Mutation Timeline

Track mutation and action calls with:
- Function name and arguments
- Success/error status
- Duration timing
- Optimistic update indicators

### Auth Panel

Inspect authentication state:
- Current user data
- JWT token claims and expiry
- Auth flow timing waterfall (SSR token exchange → client hydration → refresh)

### Connection State

Monitor the WebSocket connection:
- Connected/disconnected status
- Retry count
- In-flight requests

## Accessing DevTools

1. Start your dev server: `nuxt dev --dotenv .env.local`
2. Open Nuxt DevTools (click the Nuxt icon in the bottom of your browser, or press `Shift+Alt+D`)
3. Navigate to the **Convex** tab

## Production

DevTools are automatically disabled in production builds. There is zero runtime overhead — the DevTools code is tree-shaken from your production bundle.
