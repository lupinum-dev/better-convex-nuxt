---
title: 'Shared Schemas'
description: 'Define argument shapes once, then reuse them in Convex handlers, MCP tools, and server routes.'
---

When the same input shape is used in your Convex backend and your MCP tools or forms, define it once in a shared file. The `@lupinum/trellis/args` package provides `defineArgs(...)` for this.

## Define Once

```ts [shared/schemas/post.ts]
import { defineArgs } from '@lupinum/trellis/args'
import { v } from 'convex/values'

export const createPost = defineArgs({
  description: 'Create a post',
  args: {
    title: v.string(),
    content: v.string(),
  },
  meta: {
    title: { label: 'Title', description: 'Post title' },
    content: { label: 'Content', description: 'Post body' },
  },
})
```

## Reuse In Convex

```ts [convex/posts.ts]
import { app } from './functions'

export const create = app.mutation({
  guard: canCreatePost,
  args: createPost.args,
  handler: async (ctx, args) => {
    const actor = await ctx.actor()
    return await ctx.db.insert('posts', {
      title: args.title,
      content: args.content,
      ownerId: actor.userId,
      workspaceId: actor.tenantId,
    })
  },
})
```

## Reuse In Nitro Or MCP

```ts [server/mcp/tools/create-post.ts]
import { internal } from '#trellis/api'

import { projectTool } from '../lib/mcp-runtime'

export default projectTool({
  call: internal.posts.create,
  schema: createPost,
  capability: 'createPost',
  meta: {
    name: 'create-post',
  },
})
```

## Why `shared/` Exists

`shared/` is a runtime-boundary folder, not a Nuxt convention. Use it when both:

- `convex/` files
- Nuxt `server/` files

need to import the same args definitions.

## The `meta` Field

The `meta` object adds labels, descriptions, and examples to each argument. These are used by:

- **MCP tools** — the agent sees the description when discovering tool parameters
- **Form generation** — labels and descriptions drive auto-generated forms
- **Validation hints** — examples help agents fill in realistic values

```ts
meta: {
  title: {
    label: 'Title',
    description: 'Short title for the page',
    examples: ['Welcome to our blog'],
  },
}
```

Keep shared schemas runtime-neutral so both Convex's bundler and Nitro's bundler can load them safely.
