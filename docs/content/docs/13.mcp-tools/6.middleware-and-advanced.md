## Middleware

Middleware runs after auth and before the handler:

```ts
export default defineTool({
  schema,
  auth: 'required',
  middleware: async (_args, ctx, next) => {
    console.log(`[audit] user=${ctx.actor?.userId ?? 'anonymous'}`)
    return next()
  },
  handler: async (args, ctx) => {
    return ctx.ok({ args })
  },
})
```

Block execution by returning `wrapError(...)` instead of `next()`.

## Enabled Guards

```ts
enabled: (event) => !!event.context.mcpAuth
```

Use this to hide tools from `tools/list` when they should not appear for the current request.

`defineTool()` already hides tools when auth, `scoped`, or actor `check` rules fail. Use explicit `enabled` only for app-specific discovery rules that cannot be expressed through the built-in options.

## Operation Types

- `query`: read-only
- `mutation`: Convex write
- `action`: may call external systems

These drive the default MCP annotations.

## Tags And Group

`tags` and `group` are free-form classification metadata. Keep them human-meaningful, not over-modeled.
