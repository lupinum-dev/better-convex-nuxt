All module options live under the `convex` key in `nuxt.config.ts`:

```ts [nuxt.config.ts]
export default defineNuxtConfig({
  modules: ['better-convex-nuxt'],
  convex: {
    // options go here
  },
})
```

For env ownership, use the canonical [Environment Matrix](/docs/deployment/environment-matrix).

## Runtime Fallback Rules

The module resolves configuration from:

1. explicit `convex` options in `nuxt.config.ts`
2. runtime/public runtime config
3. env fallbacks from source

Important env behavior:

- `url` defaults to `NUXT_PUBLIC_CONVEX_URL || CONVEX_URL`
- `siteUrl` defaults to `NUXT_PUBLIC_CONVEX_SITE_URL || CONVEX_SITE_URL`
- if no explicit site URL exists, the module tries to derive it from `url`
- local Convex dev hosts like `http://127.0.0.1:3210` derive to `http://127.0.0.1:3211`

## Top-Level Options

| Option | Type | Default | Notes |
| ------ | ---- | ------- | ----- |
| `url` | `string` | `NUXT_PUBLIC_CONVEX_URL \|\| CONVEX_URL` | Primary Convex deployment URL. |
| `siteUrl` | `string` | `NUXT_PUBLIC_CONVEX_SITE_URL \|\| CONVEX_SITE_URL \|\| derived from url` | Optional HTTP actions override. |
| `auth` | `ConvexAuthConfigInput` | enabled with default route protection settings | Disable with `auth: { enabled: false }`. |
| `authRoute` | `string` | `'/api/auth'` | Normalized to always start with `/` and have no trailing slash. |
| `trustedOrigins` | `string[]` | `[]` | Extra allowed origins for the auth proxy. |
| `skipAuthRoutes` | `string[]` | `[]` | Route globs that skip auth bootstrap. |
| `permissions` | `boolean` | `false` | Enables `createPermissions` auto-import. |
| `logging` | `false \| 'info' \| 'debug'` | `false` | Module log level. |
| `debug` | `ConvexDebugOptions` | all flags `false` | High-verbosity trace channels. |
| `authCache` | `AuthCacheOptions` | `{ enabled: false, ttl: 60 }` | SSR token cache. TTL is clamped to `1..60`. |
| `query` | `QueryDefaults` | `{ server: true, subscribe: true }` | Global defaults for query composables. |
| `upload` | `UploadDefaults` | `{ maxConcurrent: 3 }` | Default upload queue concurrency. |
| `authProxy` | `AuthOptions` | `{ maxRequestBodyBytes: 1048576, maxResponseBodyBytes: 1048576 }` | Body size limits for the Better Auth proxy. |

## Authentication Defaults

The default auth config from source is effectively:

```ts
convex: {
  auth: {
    enabled: true,
    routeProtection: {
      redirectTo: '/auth/signin',
      preserveReturnTo: true,
    },
    unauthorized: {
      enabled: false,
      redirectTo: '/auth/signin',
      includeQueries: false,
    },
  },
}
```

## Query Defaults

```ts [nuxt.config.ts]
export default defineNuxtConfig({
  convex: {
    query: {
      server: true,
      subscribe: true,
    },
  },
})
```

- `server: false` disables SSR query execution by default.
- `subscribe: false` makes queries HTTP-only by default.

## Auth Cache Example

```ts [nuxt.config.ts]
export default defineNuxtConfig({
  convex: {
    authCache: {
      enabled: true,
      ttl: 60,
    },
  },
})
```

For multi-instance deployments:

```ts [nuxt.config.ts]
export default defineNuxtConfig({
  convex: {
    authCache: {
      enabled: true,
      ttl: 60,
    },
  },
  nitro: {
    storage: {
      'cache:convex:auth': {
        driver: 'redis',
        url: process.env.REDIS_URL,
      },
    },
  },
})
```

## Full Example

```ts [nuxt.config.ts]
export default defineNuxtConfig({
  modules: ['better-convex-nuxt'],
  convex: {
    url: process.env.CONVEX_URL,
    siteUrl: process.env.CONVEX_SITE_URL,
    authRoute: '/api/auth',
    trustedOrigins: ['https://preview-*.vercel.app'],
    skipAuthRoutes: ['/', '/pricing', '/docs/**'],
    permissions: true,
    logging: process.env.NODE_ENV === 'development' ? 'debug' : false,
    debug: {
      authFlow: false,
      clientAuthFlow: false,
      serverAuthFlow: false,
    },
    query: {
      server: true,
      subscribe: true,
    },
    upload: {
      maxConcurrent: 3,
    },
    authProxy: {
      maxRequestBodyBytes: 1_048_576,
      maxResponseBodyBytes: 1_048_576,
    },
    authCache: {
      enabled: false,
      ttl: 60,
    },
  },
})
```
