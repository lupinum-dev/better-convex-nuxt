Use this page as the public contributor map.

The repo-root guide remains the canonical source document: [CONTRIBUTING.md](https://github.com/lupinum-dev/trellis/blob/main/CONTRIBUTING.md)

## Local Start

Core setup:

```bash
pnpm install
pnpm run dev:prepare
pnpm run dev
```

## Main Working Areas

- `src/`: module, runtime, MCP, server, testing
- `examples/`: runnable example apps
- `test/`: unit, Nuxt, server, browser, e2e, internal harness
- `docs/`: docs app

## When Docs Should Change Too

Docs should usually change in the same PR when you change:

- public runtime behavior
- config shape
- setup flow
- permissions or trust-boundary semantics
- API surface or examples that docs rely on

## Preview And Validation

- preview docs locally with `pnpm --dir docs dev`
- validate docs links with `pnpm check:docs:links`
