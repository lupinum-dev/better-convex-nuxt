# vNext Architecture Record

This directory preserves the design specifications used for the `0.6.0` hard cutover:

- [`vNext.md`](./vNext.md) defines the approved public contract and acceptance criteria.
- [`wenext_internal.md`](./wenext_internal.md) records the internal ownership, security, testing, and release decisions.

These files are historical architecture records, not current API documentation. Maintained usage guidance lives in the repository [`README.md`](../../../README.md) and under [`docs/content`](../../content/). The vocabulary checker intentionally excludes this directory because the specifications describe APIs that were deleted during the cutover.
