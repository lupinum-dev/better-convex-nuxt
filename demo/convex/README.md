# Demo Convex Workspace

This folder contains the Convex backend used by the polished `demo/` Nuxt app.

Use it for:

- deployment-oriented example code
- auth/provider configuration examples
- schema and function patterns that are meant to mirror public docs

Key files:

- `auth.ts`: Better Auth setup and provider wiring
- `http.ts`: Better Auth route registration
- `schema.ts`: demo data model
- `permissions.config.ts`: demo permission rules

For local env and workspace setup, use [../../DEVELOPMENT.md](../../DEVELOPMENT.md).
