Reference for `trellis.mcp`. These are module-level flags — tool definitions, runtime auth, and operation binding live in [MCP: getting started](/apps/docs/mcp-tools/getting-started).

```ts [nuxt.config.ts]
trellis: {
  mcp: {
    name: 'My App',
    sessions: false,
  },
}
```

## Option reference

| Option | Default | Purpose |
| --- | --- | --- |
| `name` | `''` | Name shown to MCP clients |
| `sessions` | `false` | Enable MCP session state |

## Scope

This page is about module config, not tools. If you're defining tools, resources, or prompts, that's the [MCP tools section](/apps/docs/mcp-tools/getting-started).
