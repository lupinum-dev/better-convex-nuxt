# Public Starter

Small Nuxt + Convex starter for public apps.

## Includes

- one public `todos` table;
- realtime list query;
- create, toggle, and remove mutations;
- invariant tests for validation and list ordering.

## Non-goals

- no auth;
- no organizations;
- no MCP;
- no agents;
- no shared access package.

## Commands

```bash
pnpm install
pnpm convex:dev
pnpm dev
pnpm test
pnpm typecheck
```
